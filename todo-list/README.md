# ✅ React Todo App

A simple, functional Todo List application built with **React** and **Vite**, using `uuid` for unique task identification.

---

## 📁 Project Structure

```
src/
├── main.jsx       # React entry point
├── App.jsx        # Core app logic and UI
├── App.css        # Component-level styles
└── index.css      # Global styles and CSS variables (light/dark theme)
```

---

## ✨ Features

- **Add tasks** via text input
- **Delete tasks** individually (red ✕ button)
- **Convert all tasks** to UPPERCASE or lowercase in bulk
- **Unique IDs** for each task using `uuid`
- **Light/Dark theme** support via CSS variables and `prefers-color-scheme`
- **Responsive layout** with breakpoints at 1024px

---

## 🛠 Tech Stack

| Technology | Purpose |
|------------|---------|
| React 18 | UI framework |
| Vite | Build tool & dev server |
| uuid (v4) | Unique task IDs |
| CSS Variables | Theming (light/dark) |
